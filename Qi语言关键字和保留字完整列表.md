## 变量命名建议

### ✅ 推荐的变量名

**中文变量名（非保留关键字）：**
```qi
变量 总和 = 100;
变量 数量 = 5;
变量 名称 = "张三";
变量 苹果 = 10;
变量 用户 = 用户对象;
```

**英文变量名：**
```qi
变量 result = 42;
变量 count = 10;
变量 message = "Hello";
变量 user = userObject;
```

**驼峰命名：**
```qi
变量 myResult = 42;
变量 itemCount = 10;
```

### ❌ 不能使用的变量名

所有上述保留关键字都不能作为变量名，例如：

```qi
// ❌ 错误示例
变量 结果 = 42;      // "结果"是Result类型
变量 数字 = 100;     // 可能与类型混淆
变量 整数 = 42;      // "整数"是类型名
变量 浮点数 = 3.14;  // "浮点数"是类型名
变量 如果 = 5;       // "如果"是控制流关键字
变量 函数 = "test";  // "函数"是声明关键字
```

## 调试提示

如果遇到 "Keyword used as identifier" 或 "解析错误"：
1. 检查变量名是否在上述保留关键字列表中
2. 尝试使用英文变量名或描述性的中文变量名
3. 参考推荐命名示例

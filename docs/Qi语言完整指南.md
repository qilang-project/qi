## 概述

Qi（气）是一门 100% 使用中文关键字的编译型编程语言，通过 LLVM 后端编译为原生可执行文件。

### 语言特点

- **纯中文关键字** — 所有编程概念均用中文表达，从 `函数` 到 `如果` 到 `启动`
- **类型推断** — 编译器自动推断变量类型，无需显式声明
- **现代并发** — 基于协程的 M:N 并发模型，通道通信（类似 Go 的 CSP 模式）
- **异步支持** — `未来<T>` 类型实现异步编程
- **LLVM 编译** — 编译为高效的机器码，支持 macOS / Linux / Windows
- **丰富的标准库** — HTTP、JSON、网络、加密、GUI、数据库、WebSocket、MCP 等 25+ 模块
- **中英文标点通用** — 分号 `;` 和 `；`、冒号 `:` 和 `：` 均可使用

### 项目结构

```
qilang/
├── qi/              # Qi 编译器核心（Rust / LLVM）
├── qi-gui/          # GUI 图形界面库（Tao + Skia）
├── qi-lsp/          # 语言服务器（LSP 协议）
├── qi-tools/        # 开发工具集（qifmt 格式化器）
├── qi-vscode/       # VS Code 扩展（语法高亮、补全、主题）
├── qi-installer/    # 跨平台安装器
└── docs/            # 文档
```

## 快速入门

### 第一个程序

创建文件 `你好世界.qi`：

```qi
包 主程序;

函数 入口() {
    打印行("你好，Qi！");
}
```

运行：

```bash
qi run 你好世界.qi
```

输出：

```
你好，Qi！
```

**关键概念：**

- `包 主程序;` — 每个 Qi 文件必须声明包名
- `函数 入口()` — 程序入口点，等价于 `main()`
- `打印行()` — 输出一行文本到控制台（自动换行）
- `打印()` — 输出文本（不换行）

### 变量与计算

```qi
包 主程序;

函数 入口() {
    变量 a = 10;
    变量 b = 5;
    变量 和 = a + b;      // 15
    变量 差 = a - b;      // 5
    变量 积 = a * b;      // 50
    变量 商 = a / b;      // 2

    打印(和);
    打印(差);
    打印(积);
    打印(商);
}
```

### 条件与循环

```qi
包 主程序;

函数 入口() {
    变量 分数 = 85;

    如果 分数 >= 90 {
        打印("优秀");
    } 否则 如果 分数 >= 80 {
        打印("良好");
    } 否则 如果 分数 >= 60 {
        打印("及格");
    } 否则 {
        打印("不及格");
    }

    // while 循环
    变量 计数 = 0;
    当 计数 < 5 {
        打印(计数);
        计数 = 计数 + 1;
    }
}
```

### 函数

```qi
包 主程序;

// 带参数和返回值的函数
函数 加法(x: 整数, y: 整数): 整数 {
    返回 x + y;
}

// 浮点数计算
函数 圆面积(半径: 浮点数): 浮点数 {
    变量 圆周率 = 3.14159;
    返回 圆周率 * 半径 * 半径;
}

函数 入口() {
    变量 和 = 加法(10, 20);
    打印(和);           // 30

    变量 面积 = 圆面积(5.0);
    打印(面积);         // 78.53975
}
```

### 综合示例：简单计算器

```qi
包 主程序;

函数 加(a: 整数, b: 整数): 整数 { 返回 a + b; }
函数 减(a: 整数, b: 整数): 整数 { 返回 a - b; }
函数 乘(a: 整数, b: 整数): 整数 { 返回 a * b; }

函数 除(a: 整数, b: 整数): 整数 {
    如果 b == 0 {
        打印("错误：除数不能为零");
        返回 0;
    }
    返回 a / b;
}

函数 入口() {
    变量 x = 20;
    变量 y = 5;

    打印("加法:"); 打印(加(x, y));
    打印("减法:"); 打印(减(x, y));
    打印("乘法:"); 打印(乘(x, y));
    打印("除法:"); 打印(除(x, y));
}
```

## 安装与命令行

### 环境要求

- Rust 1.75+
- LLVM 15.0+（用于代码生成）
- Clang（用于链接）
- macOS / Linux / Windows

### 构建编译器

```bash
# 克隆项目
git clone https://github.com/qilang/qilang.git
cd qilang

# 构建所有子项目
cargo build --workspace --release

# 编译器可执行文件位于
./target/release/qi
```

### CLI 命令

所有命令均支持中英文名称：

| 命令                   | 中文别名               | 说明             |
| ---------------------- | ---------------------- | ---------------- |
| `qi run 文件.qi`       | `qi 运行 文件.qi`      | 编译并运行       |
| `qi compile 文件.qi`   | `qi 编译 文件.qi`      | 编译为可执行文件 |
| `qi check 文件.qi`     | `qi 检查 文件.qi`      | 仅检查语法       |
| `qi format 文件.qi -i` | `qi 格式化 文件.qi -i` | 格式化源代码     |
| `qi debug 文件.qi`     | `qi 调试 文件.qi`      | 调试运行         |
| `qi info`              | `qi 信息`              | 显示编译器信息   |

#### 全局选项

```bash
qi run 程序.qi \
    -t macOS           # 目标平台 (Linux, Windows, macOS, Wasm)
    -O standard        # 优化级别 (none, basic, standard, maximum)
    -o 输出文件         # 输出文件路径
    -d                 # 包含调试符号
    --verbose          # 详细输出
    --import-paths 路径 # 导入路径（冒号分隔）
```

### 类型概览

| 类型名   | 大小 | 说明         | 示例                  |
| -------- | ---- | ------------ | --------------------- |
| `整数`   | 32位 | 有符号整数   | `42`, `-100`          |

## 控制流

### if 条件语句

```qi
// 基本
如果 年龄 >= 18 {
    打印行("成年人");
}

// if-else
如果 年龄 >= 18 {
    打印行("成年人");
} 否则 {
    打印行("未成年");
}

// if-else if-else 链
如果 分数 >= 90 {
    打印行("优秀");
} 否则 如果 分数 >= 80 {
    打印行("良好");
} 否则 如果 分数 >= 60 {
    打印行("及格");
} 否则 {
    打印行("不及格");
}
```

### while 循环

```qi
变量 计数 = 0;

当 计数 < 5 {
    打印(计数);
    计数 = 计数 + 1;
}

// 无限循环
当 真 {
    如果 应该跳出 {
        跳出;
    }
}
```

### for 循环

```qi
// 遍历数组
变量 数字列表 = [1, 2, 3, 4, 5];

对于 数字 在 数字列表 中 {
    打印(数字);
}
```

### loop 无限循环

```qi
变量 计数 = 0;

循环 {
    计数 = 计数 + 1;

    如果 计数 > 10 {
        跳出;      // break
    }

    如果 计数 % 2 == 0 {
        继续;      // continue
    }

    打印(计数);    // 只打印奇数
}
```

### match 匹配

```qi
变量 值 = 42;

选择 值 {
    情况 1 => 打印行("是一"),
    情况 2 => 打印行("是二"),
    情况 42 => 打印行("是四十二"),
    情况 _ => 打印行("其他值"),
}

// 匹配范围
选择 年龄 {
    情况 0..17 => 打印行("未成年"),
    情况 18..65 => 打印行("成年"),
    情况 _ => 打印行("老年"),
}

// 匹配条件
选择 {
    情况 数字 < 10 => 打印行("小于 10"),
    情况 数字 < 20 => 打印行("小于 20"),
    情况 _ => 打印行("大于等于 20"),
}
```

### 完整示例：流程控制

```qi
包 主程序;

函数 入口() {
    变量 数字 = 10;

    // if-else
    如果 数字 > 5 {
        打印("数字大于5");
    } 否则 {
        打印("数字小于等于5");
    }

    // while 循环
    变量 计数器 = 0;
    当 计数器 < 3 {
        打印(计数器);
        计数器 = 计数器 + 1;
    }

    // 嵌套控制流
    变量 i = 0;
    当 i < 2 {
        如果 i == 0 {
            打印("第一次");
        } 否则 {
            打印("第二次");
        }
        i = i + 1;
    }
}
```

## 函数

### 基本语法

```qi
// 无参数、无返回值
函数 打印问候() {
    打印行("你好！");
}

// 带参数
函数 打印姓名(姓名: 字符串) {
    打印行("你好，" + 姓名 + "！");
}

// 带返回值
函数 加法(甲: 整数, 乙: 整数): 整数 {
    返回 甲 + 乙;
}
```

### 表达式函数

```qi
// 单表达式函数
函数 平方(数字: 整数): 整数 = 数字 * 数字;
函数 是正数(数字: 整数): 布尔 = 数字 > 0;
```

### 递归函数

```qi
包 主程序;

// 递归斐波那契
函数 斐波那契(第几项: 整数): 整数 {
    如果 第几项 小于等于 1 {
        返回 第几项;
    }
    返回 斐波那契(第几项 减 1) 加 斐波那契(第几项 减 2);
}

// 递归阶乘
函数 阶乘(数值: 整数): 整数 {
    如果 数值 小于等于 1 {
        返回 1;
    }
    返回 数值 乘 阶乘(数值 减 1);
}

// 最大公约数（欧几里得算法）
函数 最大公约数(甲: 整数, 乙: 整数): 整数 {
    如果 乙 等于 0 {
        返回 甲;
    }
    返回 最大公约数(乙, 甲 取余 乙);
}
```

### 斐波那契数列（递归 + 迭代）

```qi
包 主程序;

// 递归实现
函数 斐波那契递归(n: 整数) : 整数 {
    如果 n <= 0 { 返回 0; }
    如果 n == 1 { 返回 1; }
    返回 斐波那契递归(n - 1) + 斐波那契递归(n - 2);
}

// 迭代实现（更高效）
函数 斐波那契迭代(n: 整数) : 整数 {
    如果 n <= 0 { 返回 0; }
    如果 n == 1 { 返回 1; }

    变量 前前: 整数 = 0;
    变量 前一: 整数 = 1;
    变量 当前: 整数 = 0;
    变量 i: 整数 = 2;

    当 i <= n {
        当前 = 前前 + 前一;
        前前 = 前一;
        前一 = 当前;
        i = i + 1;
    }

    返回 当前;
}

函数 入口() {
    打印行("=== 斐波那契数列 ===");
    变量 n: 整数 = 10;
    变量 结果值: 整数 = 斐波那契迭代(n);
    打印行("第10项:");
    打印行(结果值);
}
```

```qi
导入 标准库.输入输出;

// 函数：左花括号不换行，4 空格缩进
函数 计算总和(甲: 整数, 乙: 整数): 整数 {
    变量 计算值 = 甲 + 乙;

    如果 计算值 > 0 {
        打印行("正数");
    } 否则 {
        打印行("非正数");
    }

    返回 计算值;
}

// 结构体：每行一个字段
类型 用户 {
    公开 字符串 姓名;
    公开 整数 年龄;
    私有 布尔 是否激活;
}
```

## 常见问题与陷阱

### `结果` 是保留字

```qi
// ❌ 错误：结果 是保留关键字
变量 结果 = 加法(1, 2);

// ✅ 正确：使用其他名字
变量 计算值 = 加法(1, 2);
变量 返回值 = 加法(1, 2);
变量 加法结果 = 加法(1, 2);
```

### 模块名大小写敏感

```qi
// ❌ 错误：http 是小写
导入 标准库.http;

// ✅ 正确：HTTP 要大写
导入 标准库.HTTP;
```

### 用 `.` 不用 `::`

```qi
// ❌ 错误：Qi 不使用 :: 访问模块函数
变量 响应 = HTTP::获取("...");

// ✅ 正确：用 . 分隔
变量 响应 = HTTP.获取("...");
```

### 字符串模块的名字

```qi
// ❌ 错误：字符串 是类型关键字
导入 标准库.字符串;

// ✅ 正确：用 文本 作为模块名
导入 标准库.文本;
```

### TCP连接需要3个参数

```qi
// ❌ 错误：缺少超时参数
变量 连接 = 网络.TCP连接("127.0.0.1", 6759);

// ✅ 正确：主机, 端口, 超时毫秒
变量 连接 = 网络.TCP连接("127.0.0.1", 6759, 5000);
```

### 多文件 `入口()` 冲突

> 只有**入口文件**的 `入口()` 会编译为 `@main`。被导入的模块中如有 `入口()`，它只是普通函数，不会产生重复的 `_main` 符号。

### 端口号规范

```qi
// ❌ 避免使用常见端口
变量 端口 = 8080;
变量 端口 = 3000;

// ✅ 使用随机高位端口
变量 端口 = 43510;
变量 端口 = 6759;
```

### 其他注意事项

- 所有示例文件必须以 `包 主程序;` 开头
- 入口函数是 `函数 入口()` 而非 `函数 主函数()` 或 `函数 main()`
- `打印行()` 自动换行，`打印()` 不换行
- 注释使用 `//`（单行）和 `/* */`（多行），与 C/Rust 相同
- 硬编码的地址和端口应提取为变量

### 编译流程

```
.qi 源文件 → 词法分析 → 解析(LALRPOP) → AST → 语义分析 → LLVM IR → 原生可执行文件
```

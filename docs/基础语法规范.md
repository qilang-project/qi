# Qi 基础语法规范

本文档记录当前推荐写法。解析器可能保留少量旧写法兼容，但新代码和示例应优先使用本规范。

## 总原则

- Qi 用户可见 API 优先中文。
- 外部协议值、环境变量名、模型名、URL、HTTP 方法字符串等可以保留英文。
- 类型声明、函数声明、变量声明保持 `名称: 类型` 风格。
- 旧写法只作为兼容层，不作为新示例推荐写法。

## 包与导入

```qi
包 主程序;

导入 标准库.JSON;
导入 标准库.大模型;
```

公开导出使用：

```qi
导出 响应::数据响应;
导出 路由表::获取;
```

## 变量与常量

推荐：

```qi
变量 姓名: 字符串 = "张三";
变量 年龄: 整数 = 25;
常量 最大数量: 整数 = 100;
```

允许省略类型让编译器推断：

```qi
变量 计数 = 0;
```

不推荐把关键字和变量名连在一起：

```qi
// 不推荐
变量姓名: 字符串 = "张三";
```

## 函数

推荐：

```qi
函数 加法(左: 整数, 右: 整数) : 整数 {
    返回 左 + 右;
}
```

默认参数：

```qi
函数 问候(名字: 字符串 = "朋友") : 字符串 {
    返回 "你好，" + 名字;
}
```

可变参数当前可解析，运行时语义仍应继续补测试：

```qi
函数 求和(数字...: 整数) : 整数 {
    返回 0;
}
```

## 条件与循环

```qi
如果 (年龄 >= 18) {
    打印行("成年人");
} 否则 {
    打印行("未成年人");
}
```

```qi
当 (计数 < 10) {
    计数 = 计数 + 1;
}
```

推荐：

```qi
对于 项 在 列表 {
    打印行(项);
}
```

兼容：

```qi
对于 (项 在 列表) {
    打印行(项);
}
```

## 结构体与字段

推荐字段写法是 `字段名: 类型`：

```qi
类型 用户 {
    姓名: 字符串,
    年龄: 整数
}
```

旧写法只保留兼容：

```qi
类型 用户 {
    字符串 姓名;
    整数 年龄;
}
```

新示例和新文档不应再使用旧写法。

结构体字面量当前推荐显式写：

```qi
变量 用户1 = (用户 { 姓名: "张三", 年龄: 25 });
```

## 数组

```qi
变量 数字们: 数组<整数> = [1, 2, 3];
变量 第一个: 整数 = 数字们[0];
```

数组类型和运行时能力还需要继续补齐 semantic/codegen 测试。

## JSON 与工具返回

需要返回 JSON 字符串时，优先用 `标准库.JSON`：

```qi
导入 标准库.JSON;

函数 查询天气(参数: 字符串) : 字符串 {
    返回 JSON.从键值("城市=上海;天气=晴;温度=25°C");
}
```

普通文本转 JSON：

```qi
返回 JSON.从文本("执行完成");
```

不推荐手写大量转义 JSON 字符串。

## Web API 命名

公开 API 推荐中文：

```qi
表 = 路由表.获取(表, "/", "首页#显示");
表 = 路由表.提交(表, "/用户", "用户#创建");
表 = 路由表.删除路由(表, "/用户/{id}", "用户#删除");
```

`GET`、`POST`、`DELETE` 等只作为兼容别名和 HTTP 协议值保留。

## 关键字与标识符

硬关键字和类型关键字不应作为普通变量名：

```qi
// 不推荐
变量 数组 = 1;
变量 字符 = "x";
变量 选项 = "a";
```

`长度` 允许作为普通标识符：

```qi
变量 长度: 整数 = 10;
```

字段名、方法名、模块名允许范围应由语法规则显式控制，不应靠临时特例扩展。

## 注释

支持：

```qi
// 单行注释
```

支持普通块注释：

```qi
/* 块注释 */
```

三引号注释当前不作为正式语法。

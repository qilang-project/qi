# Qi语言 命令行标准库 - 实施计划

## 📋 概览

基于Rust的clap库设计，为Qi语言提供现代化的CLI参数解析功能。

## 🎯 核心API设计

### 简单使用示例

```qi
包 主程序;
导入 标准库.命令行;

函数 入口() {
    // 1. 创建应用
    变量 应用 = 命令行.创建应用("mytool")
        .版本("1.0.0")
        .关于("我的CLI工具");

    // 2. 添加参数
    应用.添加参数(
        命令行.创建参数("文件")
            .帮助("输入文件")
            .必需(真)
    );

    应用.添加选项(
        命令行.创建选项("输出")
            .短名("o")
            .长名("output")
            .默认值("out.txt")
    );

    应用.添加标志(
        命令行.创建标志("详细")
            .短名("v")
            .长名("verbose")
    );

    // 3. 解析
    变量 匹配 = 应用.解析();

    // 4. 使用
    变量 文件: 字符串 = 匹配.获取值("文件");
    变量 输出: 字符串 = 匹配.获取值("输出");
    变量 详细: 布尔 = 匹配.获取标志("详细");
}
```

### 子命令示例

```qi
变量 应用 = 命令行.创建应用("git");

// 添加 clone 子命令
变量 克隆 = 命令行.创建子命令("clone")
    .关于("克隆仓库");
克隆.添加参数(命令行.创建参数("地址").必需(真));

应用.添加子命令(克隆);

变量 匹配 = 应用.解析();

如果 (匹配.包含子命令("clone")) {
    变量 子匹配 = 匹配.获取子命令("clone");
    变量 地址 = 子匹配.获取值("地址");
}
```

## 🏗️ 技术架构

### 三层架构

```
┌─────────────────────────────────────┐
│   Qi语言层（示例/标准库/用户代码）    │
│   - 命令行.创建应用()                │
│   - 匹配.获取值()                    │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│   模块注册层（module_registry.rs）   │
│   - 函数映射: 创建应用 -> qi_cli_...│
│   - LLVM IR声明生成                 │
└─────────────────────────────────────┘
                 ↓
┌─────────────────────────────────────┐
│   FFI层（cli_ffi.rs）               │
│   - C ABI导出函数                   │
│   - 使用clap实现核心逻辑             │
└─────────────────────────────────────┘
```

### 核心FFI函数列表

| 函数名 | 参数 | 返回值 | 说明 |
|-------|------|--------|------|
| `qi_cli_create_app` | name: *char | *void | 创建应用 |
| `qi_cli_set_version` | app, version | void | 设置版本 |
| `qi_cli_set_about` | app, about | void | 设置描述 |
| `qi_cli_create_arg` | name: *char | *void | 创建参数 |
| `qi_cli_arg_set_short` | arg, short | void | 设置短名 |
| `qi_cli_arg_set_long` | arg, long | void | 设置长名 |
| `qi_cli_arg_set_help` | arg, help | void | 设置帮助 |
| `qi_cli_arg_set_required` | arg, required | void | 设置必需 |
| `qi_cli_arg_set_default` | arg, default | void | 设置默认值 |
| `qi_cli_app_add_arg` | app, arg | void | 添加参数 |
| `qi_cli_parse` | app | *matches | 解析参数 |
| `qi_cli_get_value` | matches, name | *char | 获取字符串值 |
| `qi_cli_get_flag` | matches, name | i64 | 获取布尔标志 |
| `qi_cli_has_value` | matches, name | i64 | 检查是否有值 |
| `qi_cli_create_subcommand` | name | *void | 创建子命令 |
| `qi_cli_matches_get_subcommand` | matches, name | *matches | 获取子命令匹配 |

**预估总函数数**: 约35-40个

## 解决方案

### 1. 使用非保留关键字的中文变量名

```qi
包 主程序;

函数 获取数字() : 未来<整数> {
    返回 42;
}

函数 入口() {
    // ✅ 使用非保留关键字的中文变量名
    变量 数字: 未来<整数> = 获取数字();
    变量 值: 整数 = 等待 数字;

    打印行(值);
}
```

### 2. 使用英文变量名（推荐）

```qi
包 主程序;

函数 get_number() : 未来<整数> {
    返回 42;
}

函数 入口() {
    变量 future: 未来<整数> = get_number();
    变量 result: 整数 = 等待 future;

    打印行(result);
}
```

### 3. 混合使用（推荐）

```qi
包 主程序;

函数 获取数字() : 未来<整数> {
    返回 42;
}

函数 入口() {
    // 英文变量名用于变量，中文用于函数名
    变量 digital_future: 未来<整数> = 获取数字();
    变量 value: 整数 = 等待 digital_future;

    打印行(value);
}
```

## 最佳实践

### ✅ 推荐的变量命名

1. **英文变量名**：清晰、无歧义
   ```qi
   变量 result = 42;
   变量 count = 10;
   变量 message = "Hello";
   ```

2. **描述性的中文变量名**（非保留关键字）：
   ```qi
   变量 总和 = 100;
   变量 数量 = 5;
   变量 名称 = "张三";
   变量 苹果 = 10;
   ```

3. **驼峰命名的英文变量名**：
   ```qi
   变量 myResult = 42;
   变量 itemCount = 10;
   ```

### ❌ 避免使用的变量名

1. **保留关键字**：
   ```qi
   变量 结果 = 42;    // ❌ "结果"是Result类型
   变量 数字 = 42;    // ❌ "数字"在某些上下文中是类型
   变量 字符串 = "";  // ❌ "字符串"是String类型
   ```

2. **类型名称**：
   ```qi
   变量 整数 = 42;    // ❌ "整数"是类型名
   变量 浮点数 = 3.14; // ❌ "浮点数"是类型名
   变量 布尔 = 真;    // ❌ "布尔"是类型名
   ```

## 📊 项目概览

**Qi语言** 是一个拥有100%中文关键字的编程语言，基于Rust实现，编译到LLVM IR，支持现代编程特性。

### 标准库亮点功能

#### 1. 操作系统模块 ⭐ 最新功能

```qi
// 从 .env 文件批量加载环境变量
变量 加载数: 整数 = 操作系统.加载环境文件(".env");

// 环境变量操作
变量 值: 字符串 = 操作系统.获取环境变量("API_KEY");
操作系统.设置环境变量("KEY", "VALUE");
操作系统.删除环境变量("KEY");

// 系统信息
变量 操作系统: 字符串 = 操作系统.操作系统类型();  // "windows"/"linux"/"macos"
变量 架构: 字符串 = 操作系统.系统架构();  // "x86_64"/"aarch64"
变量 核心数: 整数 = 操作系统.CPU核心数();

// 目录操作
变量 当前: 字符串 = 操作系统.当前目录();
操作系统.切换目录("/path");
```

#### 2. 大模型模块 ⭐

```qi
// 创建LLM会话
变量 会话: 整数 = 大模型.创建会话(端点, 模型, 密钥);

// 异步调用
变量 未来: 未来<字符串> = 大模型.异步对话(会话, "问题");
变量 回答: 字符串 = 等待 未来;

// 关闭会话
大模型.关闭会话(会话);
```

#### 3. 加密模块

```qi
// 哈希计算
变量 sha256: 字符串 = 加密模块.SHA256("数据");
变量 sha512: 字符串 = 加密模块.SHA512("数据");
变量 md5: 字符串 = 加密模块.MD5("数据");

// Base64编解码
变量 编码: 字符串 = 加密模块.Base64编码("数据");
变量 解码: 字符串 = 加密模块.Base64解码(编码);
```

#### 4. HTTP模块

```qi
// HTTP GET请求

变量 结果: 字符串 = HTTP.POST(url, 数据);

// HTTP服务器
```

**最大优势**是100%中文关键字、现代异步编程支持、高性能LLVM后端，以及刚刚完成的原生 .env 文件支持。

**主要挑战**是需要完善测试、文档、工具链，以及构建更丰富的生态系统。

**建议**：
- 短期：修复测试、完善文档
- 中期：完成工具链、建立生态

项目已经具备了**生产原型的能力**，但距离**生产就绪**还有一定距离。适合用于学习、实验和原型开发，不建议直接用于关键业务系统。

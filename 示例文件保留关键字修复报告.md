# 示例文件保留关键字修复报告

## 修复概述

批量修复了所有使用保留关键字"结果"作为变量名的示例文件。

---

## 问题根源

**"结果"** 是Qi语言的保留关键字，用于 `结果<T, E>` (Result<T, E>) 类型。

### ❌ 错误示例

```qi
变量 结果 = 函数调用();  // ❌ "结果"是保留关键字
变量 md5结果 = MD5哈希();  // ❌ 包含"结果"
打印行(结果);  // ❌ 使用保留关键字
```

### ✅ 正确做法

```qi
变量 result = 函数调用();  // ✅ 使用英文变量名
变量 md5Result = MD5哈希();  // ✅ 避免保留关键字
打印行(result);  // ✅ 使用非保留关键字
```

## 修复方法

### 自动化脚本

使用 `fix_reserved_keywords.sh` 脚本批量修复：

```bash
#!/bin/bash

# 替换规则
sed -i '' 's/变量 \([^ ]*\)结果 = /变量 \1Result = /g' "$file"
sed -i '' 's/打印行(\([^)]*\)结果)/打印行(\1Result)/g' "$file"
sed -i '' 's/变量 结果 = /变量 result = /g' "$file"
sed -i '' 's/打印行(结果)/打印行(result)/g' "$file"
sed -i '' 's/返回 结果/返回 result/g' "$file"
```

### 修复模式

| 原始模式 | 修复后 | 说明 |
|---------|--------|------|
| `变量 结果 =` | `变量 result =` | 单独的"结果" |
| `变量 xxx结果 =` | `变量 xxxResult =` | 包含"结果"的变量名 |
| `打印行(结果)` | `打印行(result)` | 打印语句 |
| `返回 结果` | `返回 result` | 返回语句 |

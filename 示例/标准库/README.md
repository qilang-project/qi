# Qi 标准库示例

本目录包含 Qi 语言标准库的示例代码。

## 📁 目录结构

### 加密 (crypto)
完全可用的加密功能模块，支持：
- MD5、SHA256、SHA512 哈希
- Base64 编解码
- HMAC-SHA256 签名

**状态**: ✅ 完全可用

### 向量计算 (vector)
向量数学运算模块，用于 AI/ML 应用。

**状态**: ✅ 数组支持完成，等待 FFI 集成
- ✅ Rust 实现完成 (`src/runtime/stdlib/vector.rs`)
- ✅ FFI 绑定完成 (`src/runtime/stdlib/vector_ffi.rs`)
- ✅ 模块注册完成 (`ModuleRegistry`)
- ✅ 支持导入: `导入 标准库.向量;`
- ✅ 编译器已支持数组类型: `数组<浮点数>`
- ✅ 编译器已支持数组字面量: `[1.0, 2.0, 3.0]`
- ✅ 编译器已支持数组访问: `array[index]`
- ⚠️ 等待 FFI 调用集成（需要数组参数传递）

**已实现的 FFI 函数**:
- `qi_vector_dot` - 向量点积
- `qi_vector_add` - 向量加法
- `qi_vector_magnitude` - 向量长度(模)
- `qi_vector_normalize` - 向量归一化
- `qi_vector_cosine_similarity` - 余弦相似度
- `qi_vector_scale` - 向量数乘

**当前可用语法**:
```qi
函数 入口() {
    // 创建浮点数数组
    变量 v1: 数组<浮点数> = [1.0, 2.0, 3.0];
    变量 v2: 数组<浮点数> = [4.0, 5.0, 6.0];

    // 访问数组元素
    变量 x1: 浮点数 = v1[0];
    变量 y1: 浮点数 = v1[1];

    // 手动向量运算
    变量 点积: 浮点数 = (v1[0] * v2[0]) + (v1[1] * v2[1]) + (v1[2] * v2[2]);
}
```

**未来目标** (需要 FFI 数组传递):
```qi
导入 标准库.向量;

函数 入口() {
    变量 v1 = [1.0, 2.0, 3.0];
    变量 v2 = [4.0, 5.0, 6.0];

    // 调用 FFI 函数
    变量 点积 = 向量.点积(v1, v2);
    变量 长度 = 向量.长度(v1);
    变量 相似度 = 向量.余弦相似度(v1, v2);
}
```

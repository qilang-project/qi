- ✅ 获取用户主目录

- ✅ 系统家族 (Unix/Windows)
- ✅ 主机名

- ✅ CPU核心数
- ✅ 当前进程ID
- ✅ 退出程序

## API 参考

### 环境变量操作

```qi
// 读取环境变量
变量 值: 字符串 = 操作系统.获取环境变量("KEY");

// 设置环境变量（返回 1=成功, -1=失败）
变量 结果: 整数 = 操作系统.设置环境变量("KEY", "VALUE");

// 删除环境变量（返回 1=成功, -1=失败）
变量 结果: 整数 = 操作系统.删除环境变量("KEY");

// 列出所有环境变量（返回 KEY=VALUE 列表）
变量 所有: 字符串 = 操作系统.所有环境变量();

// 从 .env 文件加载环境变量（返回加载数量，-1=失败）
变量 数量: 整数 = 操作系统.加载环境文件(".env");
```

### 目录操作

```qi
// 获取当前工作目录
变量 目录: 字符串 = 操作系统.当前目录();

// 切换目录（返回 1=成功, -1=失败）
变量 结果: 整数 = 操作系统.切换目录("/path/to/dir");

// 获取用户主目录
变量 主目录: 字符串 = 操作系统.用户主目录();

// 获取临时目录
变量 临时: 字符串 = 操作系统.临时目录();
```

### 系统信息

```qi
// 操作系统类型 (返回: "windows" / "linux" / "macos")
变量 类型: 字符串 = 操作系统.操作系统类型();

// 系统架构 (返回: "x86_64" / "aarch64" 等)
变量 架构: 字符串 = 操作系统.系统架构();

// 系统家族 (返回: "unix" / "windows")
变量 家族: 字符串 = 操作系统.系统家族();

// 主机名
变量 主机: 字符串 = 操作系统.主机名();

// 用户名
变量 用户: 字符串 = 操作系统.用户名();

// CPU核心数
变量 核心: 整数 = 操作系统.CPU核心数();

// 进程ID
变量 进程: 整数 = 操作系统.进程ID();
```

## .env 文件格式

`.env` 文件使用简单的 `KEY=VALUE` 格式：

```env
# 注释行以 # 开头
APP_NAME=我的应用
APP_VERSION=1.0

# 支持引号（会被自动去除）
API_KEY="your-api-key-here"
DB_PASSWORD='secret'

# 空值
OPTIONAL_VAR=
```

**语法规则：**
- 每行一个键值对，格式为 `KEY=VALUE`
- 以 `#` 开头的行是注释
- 空行会被忽略
- 值两边的引号（单引号或双引号）会被自动去除
- 不支持变量展开或复杂表达式

## 使用最佳实践

### 1. 环境配置管理

**推荐做法：**

```qi
包 主程序;
导入 标准库.操作系统;

函数 入口() {
    // 加载 .env 文件
    变量 加载数: 整数 = 操作系统.加载环境文件(".env");

    // 读取配置
    变量 api_key: 字符串 = 操作系统.获取环境变量("API_KEY");

    // 使用配置...
}
```

### 2. 多环境配置

为不同环境准备不同的 `.env` 文件：

- `.env.development` - 开发环境
- `.env.production` - 生产环境
- `.env.test` - 测试环境

根据环境加载对应文件：

```qi
变量 环境: 字符串 = 操作系统.获取环境变量("ENV");
变量 env文件: 字符串 = ".env.development";  // 默认值

// 根据 ENV 变量选择配置文件
操作系统.加载环境文件(env文件);
```

### 3. 安全注意事项

⚠️ **重要：**

1. **不要提交敏感配置到版本控制**
   - 将 `.env` 添加到 `.gitignore`
   - 只提交 `.env.example` 作为模板

2. **使用 .env.example 作为文档**
   ```env
   # .env.example - 配置模板
   API_KEY=your_api_key_here
   DB_PASSWORD=your_password_here
   ```

3. **环境变量只影响当前进程**
   - 修改环境变量不会影响父进程或其他进程
   - 进程结束后修改会丢失

### 4. 错误处理

```qi
变量 加载数: 整数 = 操作系统.加载环境文件(".env");

如果 (加载数 < 0) {
    打印行("⚠️  .env 文件不存在，使用默认配置");
    // 设置默认值
    操作系统.设置环境变量("API_KEY", "default_key");
}
```
